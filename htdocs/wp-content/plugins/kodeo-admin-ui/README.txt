== Description ==

Kodeo Admin UI turns your WordPress backend into a clean and modern user interface.
All elements are well designed and arranged for the best user experience.

If you want to report a problem or maybe you have a feature request, please send an email to support@kodeo.de

If you like Kodeo Admin UI, please leave a review here on WordPress. Thank you!

= Features =

* Elegante and stylish admin theme
* Super clean user interface
* Fully compatible with all WordPress versions since 4.0
* Works well with most 3rd party plugins (no layout conflicts)
* Tested with all popular Browsers
* Lightweight code
* Lots of theme options
* Customization

== Installation ==

To do a new installation of the plugin, please follow these steps:

1. Download the 'kodeo-admin-ui.zip' file to your computer.
1. Unzip the file
1. Upload the 'kodeo-admin-ui' folder to the '/wp-content/plugins/' directory
1. Activate the plugin through the 'Plugins' menu in WordPress
