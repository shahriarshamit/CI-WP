Styles Your Code With Prism.JS, a Lightest Code Highlighter. 

== Description ==
This plugin uses the Prism.JS to Highlight & Style Code

The plugin also adds styling to the existing prism Quick tag in your WordPress text editor.

For More Detail Check This Link: [How to Beautify WordPress Code using Prism.js](http://www.techelex.org/prismjs-beautify-wordpress-code/)

*Languages Supported:* 

* Markup
* CSS
* C-like
* JavaScript
* PHP

== Installation ==
1. Upload `plugin folder` to the `/wp-content/plugins/` directory
2. Activate the plugin through the `\Plugins\` menu in WordPress

== Frequently Asked Questions ==
= #1. No code displayed on the post? =
Ans. Please parse your code using any HTML to XML parser, Use [Postify](http://code.techelex.org/postify/) to parse your code.

= #2. How to remove line numbers? =
Ans. Simple remove `class="line-numbers"` from `<pre`.

= #3. Why its support 5 languages only? =
Ans. I want the really lightweight plugin, so its support only 5 languages.

For More Question, Check [How to Beautify WordPress Code using Prism.js](http://www.techelex.org/prismjs-beautify-wordpress-code/)
